Cara instalasi:
1. Clone project
2. Pindahkan folder flutter-server ke htdocs
3. Import database db_kamus.sql (nama database db_kamus)
4. ganti host di coding dengan ip perangkat (yang terhubung dalam satu jaringan)
5. Run project
